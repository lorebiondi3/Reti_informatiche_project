# Reti_informatiche_project
Client-server project developed for the "Reti informatiche" course of the bachelor degree in CE at University of Pisa

## Project specifications
It is required to design and implement a distributed client-server application that reproduces the Lotto game. 
The application must allow various client hosts (players) to make bets and send them to a remote server, the task of which is to store the bets, 
carry out the extractions and process the winnings, notifying the amount to the players.
